h1. NormGatewayListener

This is a simple (incomplete) module for consuming NORM ("http://cs.itd.nrl.navy.mil/work/norm/":http://cs.itd.nrl.navy.mil/work/norm/ ) messages and putting them on the JBoss ESB.

h2. Requires

The norm.*.jar library and the Eclipse JBoss Tools setup.
